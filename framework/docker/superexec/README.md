  `amd64`, `arm64v8`

## Supported tags

- `latest`
  - points to `1.21.0` and `1.21.0-py3.12-ubuntu24.04`
  - uses Python 3.12 and Ubuntu 24.04
- `unstable`
  - points to the last successful build of the `main` branch
- `nightly`, `<version>.dev<YYYYMMDD>` e.g. `1.21.0.dev20250815`
  - uses Python 3.12 and Ubuntu 24.04
